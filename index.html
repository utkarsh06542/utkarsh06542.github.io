How to use :

Tested for the following titles :

  • "Delhi" , "London" , "Mumbai"

Additional features:

  • Animated Loading symbol installed.
  • Can load article-contents from url/links in the given page/article.
  • Outputs error message in the user specified language.
  • Redirects to the best suitable title (if possible).
